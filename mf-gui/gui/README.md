# MF-GUI

### What is MF-GUI?

This is a library that simplifies creating inventory GUIs.

### **Features** <a id="why-mf"></a>

* Simple to use.
* Allows you to create updating titles, updating items or the whole GUI.
* Per item action.
* Slot specific actions.
* Close and Open actions.
* Pagination GUI.
* Scrolling GUI.
* Item Builder.
