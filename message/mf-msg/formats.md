# Formats

These is a list with the available formats that can be disabled:

* BOLD
* ITALIC
* STRIKETHROUGH
* UNDERLINE
* OBFUSCATED
* COLOR
* HEX
* GRADIENT
* RAINBOW
* ACTION\_HOVER
* ACTION\_COMMAND
* ACTION\_URL
* ACTION\_CLIPBOARD
* ACTION\_SUGGEST

If you want any other format, let me know and I'll add.
